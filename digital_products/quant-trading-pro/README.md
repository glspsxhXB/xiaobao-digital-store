## 🎯 Quick Start

```python
from main import TradingBot, Config

# Configure
config = Config(
    api_key="your_key",
    api_secret="your_secret",
    symbol="BTCUSDT"
)

# Initialize and run
bot = TradingBot(config)
await bot.run()
```
